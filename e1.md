## Números palíndromes
Dado un número entero `x`, devuelve `True` si `x` es un número palíndrome o `False` si no lo es.
(Palíndrome es que lee igual al revés).

Ejemplo 1
```python
Entrada: x = 121
Salida: True
```

Ejemplo 2
```python
Entrada: -121
Salida: False
Explicación: -121 al revés es 121-, no es palíndrome
```

Ejemplo 3
```python
Entrada: 10
Salida: False
```
